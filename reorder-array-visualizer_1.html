ReOrder Array — 四等分 A B C D，三次反转换个位

目标是把数组交替排列成 L1,R1,L2,R2,...。诀窍是：把当前段切成四等分 A B C D， 用三次反转把 B、C 换个位置变成 A C B D，再递归处理左边 [A,C] 和右边 [B,D]。

为什么三次反转能换位

      

        当前段是 [A][B][C][D]，想要变成 [A][C][B][D]（只交换中间的B、C）：
        
    
          
  1. reverse(B) — B自己整体倒过来
    2. 
          
  2. reverse(C) — C自己整体倒过来
    3. 
          
  3. reverse(B+C 合起来) — 把倒过的B、C一起再倒一次，就变成了正序的 C、B
    4. 
        

        换位之后 [A,C] 和 [B,D] 长度相等，交替排列可以各自独立完成，
        于是递归处理这两半，直到段长度 ≤ 2（天然已经是交替好的）。
